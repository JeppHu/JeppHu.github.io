GCD Time

Written by Jepp Hu with ♥ on March 15, 2018 in Foundation

延迟执行

// 参数1: 延迟多少时间, 精度到 纳秒
// 参数2: 队列
// 参数3: 任务
double second = 3.0;
dispatch_after(dispatch_time(DISPATCH_TIME_NOW, (int64_t)(second * NSEC_PER_SEC)), dispatch_get_main_queue(), ^{
    NSLog(@"延迟 %f", second);
});
