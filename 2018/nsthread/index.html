NSThread

Written by Jepp Hu with ♥ on March 15, 2018 in Foundation

多线程

多线程是切换执行的, 切换的速度非常快, 接近于 “同时” 执行, 实际上每个线程执行一段时间片就切换

程序执行过程中系统会生成一些辅助线程, 且会自动销毁

NSThread 生命周期需要程序员管理

NSThread 的使用方式

方式 1

//
NSThread *thread1 = [[NSThread alloc] initWithTarget:self selector:@selector(demo:) object:@"obj"];
// 线程名字
thread1.name = @"t1";
// 线程的优先级,默认 0.5, 取值范围 (0 ~ 1), 可以使被分配时间片的几率更大, 但是不能保证一定先执行完
thread1.threadPriority = 1;
// 线程进入可执行状态, 等待被调度, 执行
[thread1 start];

方式 2

[NSThread detachNewThreadSelector:@selector(demo) toTarget:self withObject:nil];

方式 3

[self performSelectorInBackground:@selector(demo) withObject:nil];

调用方法

- (void)demo:(NSString *)sender {
    NSLog(@"name : %@", name);
    for (int i = 0; i < 20; i ++) {
        NSLog(@"%@", [NSThread currentThread]);
        // 阻塞线程
        [NSThread sleepForTimeInterval:1];
    }
    [NSThread exit];
}
