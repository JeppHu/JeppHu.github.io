/usr/bin 和 /usr/local/bin

Written by Jepp Hu with ♥ on April 2, 2018 in Mac

usr 指 Unix System Resource，而不是 User

/usr/bin 下面的都是系统预装的可执行程序，会随着系统升级而改变

/usr/local/bin 目录是给用户放置自己的可执行程序的地方，不会被系统升级而覆盖同名文件

两个目录下有相同的可执行程序，谁优先执行受到 PATH 环境变量的影响
