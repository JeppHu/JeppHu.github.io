Git配置多账户

@Jepp Hu · Feb 21, 2019 · 1 min read

1. 为不同的账号添加对应的key

生成一个github的key

ssh-keygen -t rsa -C "youremail@domain.com"

执行命令后的第一次输入，输入一个用于区分的自定义名字,不要有空格，如：my_github_rsa， 回车 第二次与第三次的输入是为了设定访问key的密码，可以

生成一个bitbucket的key

ssh-keygen -t rsa -C "youremail@domain.com"

执行命令后的第一次输入，输入一个用于区分的名字，如：my_bitbucket

2. 修改配置文件

#进入.ssh目录，如果目录下没有config文件，那么创建
cd ~/.ssh
touch config

#添加以下内容
Host github.com
    HostName github.com
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/my github
