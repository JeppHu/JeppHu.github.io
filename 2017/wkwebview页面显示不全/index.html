WKWebView 页面显示不全

Written by Jepp Hu with ♥ on March 28, 2017 in Cocoa

WKWebView 因为 NSWindow 属性造成的 bug

NSWindowStyleMask  ==  NSWindowStyleMaskFullSizeContentView

此时 WKWebView 如果不是充满整个 Window 加载的 HTML 会出现页面显示不全的情况

解决方法

NSWindowStyleMask  ==  NSWindowStyleMaskBorderless
