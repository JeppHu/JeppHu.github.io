Apache 服务启动和停止

Written by Jepp Hu with ♥ on June 28, 2016 in Tools

启动：sudo apachectl start

停止：sudo apachectl stop

重启：sudo apachectl restart

查看 Apache 版本 httpd -v

打开 http://127.0.0.1 可以看到 It works! 的页面，启动成功
