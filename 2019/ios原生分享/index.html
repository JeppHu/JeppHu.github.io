iOS 原生分享

Written by Jepp Hu with ♥ on February 27, 2019 in UIKit

优点：使用简单，不用注册分享平台账号，不用导入的 SDK 包

缺点：只能使用系统提供的固定样式，分享的类型受限，只能分享 text、url、image

NSArray *arr = @[result];
UIActivityViewController *activityVC = [[UIActivityViewController alloc] initWithActivityItems:arr applicationActivities:nil];
activityVC.completionWithItemsHandler = ^(UIActivityType  _Nullable activityType, BOOL completed, NSArray * _Nullable returnedItems, NSError * _Nullable activityError) {
    if (completed) {
        NSLog(@"share completed");
    } else {
        NSLog(@"share failed");
    }
};
[self presentViewController:activityVC animated:YES completion:nil];

arr 支持 UIActivity 类型，可以创建自定义的 Activity，同时分享 NSString，UIImage，NSURL
